# Benchmarks

End-to-end benchmarking is implemented by
[`scripts/user_value_benchmark.mjs`](../scripts/user_value_benchmark.mjs). The
bounded release ladder is
[`scripts/linux_canary_ladder.mjs`](../scripts/linux_canary_ladder.mjs).

This directory contains no Rust microbenchmarks. Methodology and claim limits
are documented in [`docs/benchmark.md`](../docs/benchmark.md).
